# [창의 과제] log_watchdog.py — 실시간 로그 이상 탐지

실습1은 로그를 "다 읽은 뒤" 집계하는 방식이었는데,
실무 모니터링은 흘러가는 중간에 이상을 잡아야 한다고 생각해서 만들어봤습니다.

- web_logs.csv 20만 행을 실시간 스트림처럼 재생
- 최근 300건 슬라이딩 윈도우로 5xx 비율 상시 감시 (deque 활용)
- 평소 8%보다 튀는 순간(12%↑) ALERT + 해당 구간 5xx 집중 경로 스냅샷
- 알람 폭탄 방지용 쿨다운 적용
- 종료 후 시간대별 트래픽 ASCII 차트 출력

실행: `python Advanced/log_watchdog.py`
